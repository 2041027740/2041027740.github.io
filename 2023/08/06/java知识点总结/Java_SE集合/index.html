Java集合总结

发表于 2023-08-06 更新于 2023-08-18 分类于 java知识点总结 阅读次数：
真正的热爱计算机技术，才能走得更远

集合

——————————————

Java中集合分为两大类

  • Collection：接口，使用：”父类引用指向子类对象“
  • 用来存储单列数据，例如：一些学生数据，一些字符串，一些数字……

  • Map：接口

    • 用来存双列数据（键值对数据），需要根据一个Key，快速的找到一个Value。

介绍Collection

——————————————

Collection：接口

  • 特点
    1. 用来存储单列数据
    2. 一些子实现类有序，一些无序
      • 有序：List、Queue、LinkedHashSet
      • 无序：HashSet、
      • 特殊：TreeSeta是大小有序的
    3. 一些子实现可以存重复元素，一些不可以存重复元素
      • 可存重复：List、Queue
      • 不可存重复：Set
    4. 一些可以存null ,一些不可以存null
      • 可以存null：List、HashSet、LinkedHashSet、LinkedList
      • 不可以存null：TreeSet、Queue(除了LinkedList)
  • API
    1. 增删改查API
    2. 遍历API
      • toArray()
        无参：**toArray()，拷贝一份数据，创建新的数组
        单参：        toArray(T[] t)**，传入一个数组
      1. 数组长度 < 集合长度，只将数组的数据类型传过来，不使用该数组
      2. 数组长度 = 集合长度，使用该数组
      3. 数组长度 > 集合长度，使用该数组，t[size] = null，集合存储完了之后，将数组末尾置空

      • iterator()

        里面不保存元素
        调用iterator返回的是一个接口（Iterator）

        • 调用hasNext()判断是否有下一个元素
        • 如果有调用next()，指针后移，返回刚刚经过的元素
        • 调用remove()，删除刚刚经过的元素

        注意：并发修改异常，使用迭代器的时候，不允许修改原有集合，会出现并发修改异常

      • foreach
        底层是iteator
        注意：在foreach中不要改变Collection的结构

    3. 其他API（辅助）

  • Collection子接口

    1. List接口：链表

      • 存储有序，可以存重复元素，可以存null

      • 子实现类

        1. Arraylist：后面会有一篇博客详细讲解
        2. Vector：在jdk1.2，被ArrayList替代
          vector–>子类Stack(栈)不用–>使用Deque
        3. LinkedList：链表
    2. Set接口：集合

      TreeSet 底层实现：TreeMap

      HashSet 底层实现： hashMap

      LinkedHashSet 和比HashSet多了个有序

    3. Queue接口：队列

      • 两组API

        极端情况下（插入时队列满、删除/获取时队列空）

      • 子实现接口

        • Deque(双端队列)–>子类ArrayDeque

        • 数据结构表现：队列、栈、双端队列

        • 四组API

      • 子实现类

        BlockingQueue

介绍Map

——————————————

Map：接口

  • 特点
    1. 存储双列数据（键值对），可以用一个Value快速找到一个Key
    2. 一些子实现类有序，一些无序
      • 有序：LinkedHashMap，TreeMap 大小有序
      • 无序：HashMap ，Hashtable
    3. 都不允许存储重复元素（键）
    4. 一些可以存null ， 一些不可以存null
      • 可以存null：hashMap，Hashtable，LinkedHashSet
      • 不可以存null：treeMap
  • API
    1. 增删改查
    2. 辅助API
    3. 遍历API
    4. 遍历Map的方式
      • 使用entrySet()
      • 使用keySet()，遍历keySet，拿到Key，通过get再拿到value
  • 子实现类

    HashMap：后面会有一篇博客详细讲解

    LinkedHashMa–>相对于HashMap,多了一个双向指针，来保证存储和读取数据有序

    TreeMap–>红黑树，存储数据大小有序，左小右大。
